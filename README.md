## How to Run

1. Clone the repo:
   ```bash
   git clone https://github.com/franciscpime/NHL_CS_3rd_Period.git
   ```

2. Go into any week folder and run the script:
   ```bash
   cd week1
   python main.py
   ```

> Make sure you have Python 3 installed!
